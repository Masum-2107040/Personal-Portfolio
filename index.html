About Me

Cyber Security Enthusiast

I'm Masum Molla, a Computer Science and Engineering student at KUET with a strong interest in cybersecurity—especially web application security. I'm currently focused on identifying and exploiting common web vulnerabilities such as XSS, SQL injection, CSRF, SSRF, and authentication bypass techniques. To build my skills, I actively practice on PortSwigger Web Security Academy, where I’ve been completing hands-on labs to understand the inner workings of real-world web attacks and how to defend against them. I also explore custom payload crafting, HTTP manipulation, and browser-based exploit scenarios to deepen my practical knowledge. My goal is to become a skilled ethical hacker with a solid understanding of how modern web applications can be attacked—and more importantly, how they can be secured. I'm passionate about security research, bug bounty hunting, and contributing to a safer internet.
